Introduction to HTML TAGS

HTML tags are the building blocks of a website, used to define the structure and content of a web page . They are represented by a keyword or abbreviation surrounded by angle bracket

Types of HTML TAGS

HTML tags can be categorized into two main types: 1. Container Tags (also known as Paired Tags): - Have an opening tag and a closing tag - Surround content to apply formatting or meaning - Examples: p ,div ,h1 ,ul ,table . 2. Empty Tags (also known as Singleton Tags or Self-Closing Tags): - Do not have a closing tag - Standalone tags that perform a specific function - Examples: img, , hr, input , link
